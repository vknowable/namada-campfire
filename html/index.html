LUMINARA "CAMPFIRE" NAMADA TESTNET

Chain id: CHAIN_ID
Version: NAMADA_TAG
Purpose: Short-term testnet
Peer: PEER




  
Commands to join:

  

  # set custom configs server
  export NAMADA_NETWORK_CONFIGS_SERVER="https://testnet.DOMAIN/configs"
  
  # join network
  # if your base_dir is not $HOME/.local/share/namada adjust this command accordingly
  namadac utils join-network --chain-id CHAIN_ID --wasm-dir $HOME/.local/share/namada/CHAIN_ID/wasm
  
  # add persistent peer (adjust for base_dir if needed)
  sed -i 's#persistent_peers = ".*"#persistent_peers = PEER#' $HOME/.local/share/namada/CHAIN_ID/config.toml
  
  # ...and you're ready to start your node!
  namadan ledger run
